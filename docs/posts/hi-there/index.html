Hi There

We just proceeded with CityMayor's soft launch on the weekend of March 2nd, 2018. It's been exciting and we've spent a lot of time polishing things and fixing bugs. Fortunately after weeks and weeks of testing, bugs are now only front end bugs and the smart contract at the foundation of this project remains solid.

If you're reading these words, it's probably because CityMayor ended up being a success or because you're an early beta user of the CityMayor world.

What is CityMayor? It's an application that mixes assets (cities) with fungible tokens (CityCoins). The cities can be collected and traded for different amounts of ether while the CityCoins can be exchanged like any other ERC-20 Ethereum tokens. You heard it right! by holding cities you will receive tokens that you will be able to trade on any exchanges that will support it.

To learn more about the novel concepts around CityMayor, check the whitepaper here!

Here is our roadmap for the upcoming future:

  • go through one week of beta (right now) and kill all the bugs
  • release more cities
  • official launch where new cities will see an increase in price
  • get the CityCoin token accepted on trading exchanges
  • implement monuments to increase the values of CityMayor's cities

If you find any bugs, make sure to tell us! We'll reward you with CityCoins if you send us your address :) we are reachable by email, on twitter, on facebook, on discord and on telegram.

And thank-you for participating in this exciting experiment!
